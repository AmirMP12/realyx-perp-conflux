ZERO FRONT-RUNNING

Two-phase MEV-resistant execution

Three steps from intent to position — fully on-chain, fully non-custodial

01 Phase 1 · Trader

Submit Intent

Trader calls createOrder. Collateral is locked in TradingCore. No price is consumed — front-running is structurally impossible.

solidity createOrder(CreateOrderParams params)
02 Off-chain · Keeper

Keeper Fetches Price

A Keeper node detects the pending order, fetches the latest signed price from the Pyth Hermes API, and prepares the atomic execution payload.

pyth pyth.getPriceNoOlderThan(feedId, maxAge)
03 Phase 2 · Settlement

Atomic Execution

Oracle update + order execution in one transaction. Position minted as ERC-721 NFT. Live PnL streams via WebSocket in <3 seconds.

solidity executeOrder(orderId, priceUpdateData)

SECURITY FIRST

Built to resist attacks

Every design decision in Realyx starts with the question: how could this be exploited?

Two-Phase Commit

Traders submit a signed intent via createOrder. No price is consumed at submission time. A Keeper atomically updates the oracle and executes in one transaction — front-running and sandwich attacks are structurally impossible.

Oracle Validation

Every Pyth price update is validated against block.timestamp. Stale updates revert immediately. OracleAggregator requires a minimum number of valid price sources before any order executes.

Parametric Safety

Smart contracts enforce max leverage, min collateral, position size ceilings, and liquidation thresholds. HealthLib continuously monitors protocol solvency. An Insurance Fund absorbs bad debt before the main vault is affected.

FAQ

Common questions

What is Realyx?

Realyx is a decentralized perpetual futures exchange built on Conflux eSpace. It lets you trade Crypto, Equities, and Commodities from your own wallet — no KYC, no deposits to a centralized custodian, no business-hours restrictions. Every trade is executed on-chain via a two-phase MEV-resistant architecture powered by Pyth Network oracles.

How do I get testnet USDT0 to start trading?

Connect MetaMask to Conflux eSpace Testnet (Chain ID: 71, RPC: https://evmtestnet.confluxrpc.com). Get free CFX from the Conflux faucet ↗ for gas. Then open the app, go to Settings → Testnet Tools, and click Mint 1k Mock USDT0. You're ready to trade.

Has Realyx been audited?

Realyx was built for the Conflux Global Hackfest 2026 and has not yet undergone a formal third-party security audit. The codebase includes Hardhat test cases covering core logic, fuzz testing, security scenarios, and end-to-end flows. A full independent audit is planned for Phase 2 before mainnet launch.

How do I follow Realyx and reach the team?

Updates land on X (@Realyx_Perp) and the Telegram announcements channel. For chat, questions, and feedback, join the Telegram community group. For code-level issues, open a ticket on GitHub.
